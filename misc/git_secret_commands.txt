All necessary git-secret commands you need to know.
Since git-secret is already set up you just need to know this command:

# decrypt .env file
git-secret reveal



Other commands I used for setting up git-secret:

# initialize git-secret will create a .git-secret directory
git-secret init

# Add the first user to the git-secret repo keyring
# trance@dance.com is an already existing gpg key
git secret tell trance@dance.com

# add files you wish to encrypt inside the git-secret repository
git secret add <filenames...>

# encrypt the files you added with 'git secret add'
git secret hide

# add the git secret hide command to the pre-commit hook
# which will automatically encrypt the .env file before each 'git commit'

echo 'git secret hide' > .git/hooks/pre-commit
